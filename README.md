# virtual_drum_kit

This project is an interactive virtual drum kit implemented using HTML, CSS, JavaScript, and Jupyter Notebook.

## Features

- Play drum sounds by clicking drum pads
- Play drum sounds using keyboard inputs
- Visual highlight animation when each pad is triggered
- 9 drum sounds: Bell, Crash, Ride, Hi-hat, Snare, Kick, High Tom, Middle Tom, Low Tom
- A standalone HTML version is available for web deployment

## Keyboard Mapping

| Key | Sound |
|---|---|
| Q | Bell |
| W | Crash |
| E | Ride |
| A | Hi-hat |
| S | Snare |
| D | Kick |
| Z | High Tom |
| X | Middle Tom |
| C | Low Tom |

## Files

- `index.html`: Web version for GitHub Pages
- `virtual_drum_kit.ipynb`: Jupyter Notebook source code
- `sounds/`: Drum sound files used in the notebook version

## How to Run

### Web version

Open the GitHub Pages link or open `index.html` in a browser.

### Jupyter Notebook version

Run `virtual_drum_kit.ipynb` in Jupyter Notebook or VS Code.  
The notebook reads audio files from the `sounds/` folder and generates a standalone HTML drum kit.
